### Restriction of Break Point

```
growth function mH(N): N個點時，max number of dichotomies
當mH(k) < 2^k 時，k稱為 break point (k+1, k+2,...也是break point)
當mH(k) = 2^k 時，Hypothesis Set 可以被shattered(k個點shatter H)
```
